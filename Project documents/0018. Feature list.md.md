## F001 建立分類帳戶

- **使用者故事**  
  作為使用者，我想建立一個帳戶，以便記錄金流和分類不同資金來源。

- **展示層元件**  
  - 「＋ 新增帳戶」按鈕  
  - 輸入欄位：帳戶名稱、初始餘額  
  - 下拉選單：帳戶類型（銀行、現金、電子支付）

- **使用者操作流程**  
  1. 點擊「新增帳戶」按鈕  
  2. 填寫帳戶名稱與金額  
  3. 選擇帳戶類型  
  4. 按下「建立」送出表單  
  5. 成功後跳轉回帳戶列表

- **API 路徑**  
  `POST /accounts/create`

- **API 傳送格式**
  ```json
  {
    "name": "A銀行",
    "initial_balance": 20000,
    "type": "bank"
  }

- API 回傳格式

{
  "status": "success",
  "account_id": "acc_001"
}

- 成功畫面反饋
- 顯示提示訊息：「帳戶建立成功」，並自動返回帳戶列表畫面

- 錯誤處理
  - 帳戶名稱已存在 → 顯示紅字：「帳戶名稱已存在，請更換」
  - 初始餘額輸入非數字 → 顯示：「請輸入正確金額」

- 備註
  - 幣別目前固定為 TWD（新台幣）
  - 未來可加入「帳戶圖示」選擇功能
