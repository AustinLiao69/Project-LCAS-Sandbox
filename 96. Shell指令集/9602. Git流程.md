
# 9602. Git流程

## 常用指令


## 01. **合併你的branch到main（請替換成你實際的branch名稱）：**

   ```bash
   git merge Sandbox
   ```
   
## 02. 強制重置到main(以main內容覆蓋)
   
   ```bash
   git reset --hard main
   ```
   
## 03. 比較兩個branch的差異

```bash
git diff main sandbox
```

## 04. 刪除branch

```bash
git branch -d 科目與支付方式歸類邏輯一次性互動方案
```
