
# 9602. Git流程

## 常用指令

需要替換 `your-branch-name` 為你實際的branch名稱。

### 確認現在在哪個branch
```bash
git branch
```
### 檢查當前狀態
```bash
git status
```
1. **切換到main branch：**
   ```bash
   git checkout main
   ```

2. **拉取最新的main branch變更：**
   ```bash
   git pull origin main
   ```

3. **合併你的branch到main（請替換成你實際的branch名稱）：**
   ```bash
   git merge Sandbox
   ```

4. **推送到remote main branch：**
   ```bash
   git push origin main
   ```
5. 強制push到github
```bash
git push -f your branch
```
6. 強制重置到main(以main內容覆蓋)
   git reset --hard main
   
## 01. 比較兩個branch的差異
```bash
git diff main sandbox
```

## 02. 刪除branch
```bash
git branch -d 科目與支付方式歸類邏輯一次性互動方案
```
