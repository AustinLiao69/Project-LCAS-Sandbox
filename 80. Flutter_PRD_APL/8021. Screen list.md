## 1. 簡介

此文件整合 LCAS 2.0 所需的所有畫面，基於 [0017. User story.md](./0017.%20User%20story.md) 的 38 個 User Story 和 [0019. User Flow.md](./0019.%20User%20Flow.md) 的詳細流程設計。提供**產品設計 → UI/UX 設計 → 開發規劃**的完整橋樑，確保四模式差異化體驗的實現。

---

## 2. 文件範圍 (Scope)

### 2.1 涵蓋範圍
- **進入階段**: LINE OA 加入、APP 註冊登入、跨平台整合（Epic 1）
- **記帳功能**: LINE OA 快速記帳、APP 完整記帳、記錄管理（Epic 2）
- **管理功能**: 帳本、帳戶、科目、預算管理（Epic 3-6）
- **分析功能**: 報表生成、深度分析、AI 助理（Epic 7-9）
- **系統功能**: 激勵、資料管理、安全、通知、設定、幫助（Epic 10-15）

### 2.2 不涵蓋範圍
- 後台管理功能
- 系統維護操作
- 第三方服務整合介面

### 2.3 四模式差異化策略
- **專家模式(Expert)**: 完整功能、專業工具、深度設定
- **慣性模式(Inertial)**: 簡潔介面、固定流程、一致體驗
- **養成模式(Cultivation)**: 習慣引導、進度追蹤、激勵機制
- **引導模式(Guiding)**: 極簡介面、被動引導、最少決策

## 4. 畫面詳細說明 (Screen Descriptions)

### 4.1 核心進入流程

#### S-103 APP 註冊頁
- **對應 User Story**: US-003
- **功能**: 使用者 Email 註冊與密碼設定
- **主要元件**:
  - Email 輸入框（格式驗證）
  - 密碼輸入框（強度檢查）
  - 確認密碼輸入框
  - 註冊按鈕
  - 切換至登入連結
- **四模式差異**:
  - **Expert**: 顯示完整表單，提供進階設定選項
  - **Inertial**: 簡潔表單，固定的視覺布局
  - **Cultivation**: 增加動機引導文案和進度指示
  - **Guiding**: 極簡表單，最少必要欄位

#### S-203 APP 記帳表單頁
- **對應 User Story**: US-010, US-011, US-012
- **功能**: 完整記帳資訊輸入與設定
- **主要元件**:
  - 金額輸入框（數字鍵盤）
  - 收支切換開關
  - 科目選擇按鈕
  - 帳戶選擇按鈕
  - 帳本選擇按鈕
  - 日期選擇器
  - 備註輸入框
  - 圖片附加按鈕
  - 重複設定按鈕
  - 儲存按鈕
- **四模式差異**:
  - **Expert**: 所有欄位並列顯示，支援快速輸入
  - **Inertial**: 分步驟引導，固定的填寫順序
  - **Cultivation**: 提供輸入建議和完成度指示
  - **Guiding**: 只顯示金額和基本科目，其餘自動填入

#### S-108 APP 下載推廣頁
- **對應 User Story**: US-002
- **功能**: 透過 Rich menu 推廣 APP 下載
- **主要元件**:
  - APP Store 下載按鈕
  - Google Play 下載按鈕
  - APP 功能特色介紹
  - 完整記帳功能說明
  - 略過按鈕
  - 返回 LINE OA 連結
- **四模式差異**:
  - **Expert**: 詳細功能對比表，突出專業分析工具
  - **Inertial**: 簡潔功能說明，強調一致體驗
  - **Cultivation**: 強調習慣養成和激勵機制
  - **Guiding**: 極簡介紹，重點突出易用性

### 4.2 管理功能核心

#### S-303 協作管理頁
- **對應 User Story**: US-018, US-020
- **功能**: 協作帳本的成員與權限管理
- **主要元件**:
  - 協作者列表
  - 權限角色標示
  - 邀請新成員按鈕
  - 權限編輯選單
  - 移除成員功能
  - 操作歷史記錄
- **四模式差異**:
  - **Expert**: 細粒度權限控制（檢視/編輯/管理/刪除）
  - **Others**: 基本權限設定（成員/管理員）
